# dload_cdip
MATLAB toolkit to download CDIP data directly from THREDDS server. 

Use dload_EX.m for tutorial. Otherwise, for fastest implementation:
```
[cdip] = dload_cdipbuoy(067, [datenum(2023,1,1) datenum(2023,2,1)], 1)
```
